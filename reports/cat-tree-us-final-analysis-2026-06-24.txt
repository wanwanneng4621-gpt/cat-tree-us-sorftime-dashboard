# Cat Tree 美国市场最终版选品分析报告

## Executive Summary

- **品线值得开发，但不能再做无差异的普通灰色猫爬架。** Amazon Cat Trees 实时类目 Top100 月销量为 133,229，但第三方类目流仍混有墙装猫家具；可审计的清洗明细为98条完整 Cat Tree，月销量 130,293、月销售额 $6,828,164.70。
- **首要方向是44-54英寸稳定型主流款，第二方向是60-72英寸大猫加强款。** 前者覆盖Amazon、Walmart与Temu的最大共同需求，后者承担Amazon高客单和Walmart大猫场景。
- **不同平台不能用同一SKU、同一价格和同一页面打法硬推。** Amazon卖“证据与信任”，Walmart卖“家庭价值与交付便利”，Temu卖“低价完整功能”，TikTok Shop卖“真实演示与视觉内容”。
- **美国用户最核心的不是高度，而是可用性。** 335条代表ASIN负评显示，尺寸/大猫适配、安装缺件、稳定安全、材料耐久和猫是否愿意使用，是必须优先解决的开发问题。
- **季节性不是强单峰，但存在明确的7月与11—12月抬升。** Sorftime完整月份标准化指数显示12月128.7、7月116.7、11月113.4；2026年Q4主窗口应在8月上旬至10月上旬完成分批上架，净新结构若错过开发周期则转向2027年4—5月上架、承接7月次峰。

## 1. 分析边界与数据可信度

本报告只统计落地式完整 `Cat Tree / Cat Tower / Cat Condo`。独立猫抓板、独立抓柱、猫玩具、宠物床和墙装踏板套装全部排除；抓柱和抓板仅在它们是完整 Cat Tree 的组成部件时作为功能需求分析。

- Amazon实时信号：2026年6月24日 Sorftime MCP `category_report` 与 `keyword_detail`。
- Amazon结构分析：2026年6月23日100条商品明细中剔除2条墙装产品，保留98条。
- Walmart：2026年6月17日40条完整 Cat Tree 样本。
- Temu：2026年6月24日逐个 ProductId 复核的8条完整 Cat Tree，月销量合计 9,401。
- TikTok Shop：当前没有经过商品级清洗的完整 Cat Tree 样本，因此不输出销量、市场容量或排名，只做渠道行为与内容适配推断。

## 2. 市场容量、竞争与新品趋势

### Amazon：需求最大，但主词噪音严重

- `cat tree` 月搜索量 825,293，CPC $0.82。
- `cat tree tower` 月搜索量 127,790，CPC $0.74。
- `cat tree for large cats` 月搜索量 26,889，CPC $0.92。
- 关键词搜索结果会混入独立抓板、抓柱和玩具，不能用搜索结果商品总数直接代表 Cat Tree 竞争规模。
- 实时头部出现2026年新上架的 Sweetcrispy 44/54英寸款，其中一款仅27条评论但月销量估算5,566，说明新品仍可切入，但必须在价格、视觉和供给效率上明显优于普通款。
- Amazon实时粉色54英寸款进入头部，说明审美差异不再只是低销量装饰方向，但应建立在成熟结构之上。

### Walmart：更适合家庭价值型完整猫爬架

- `cat tree` 搜索量 225,965，均价 $41.04。
- `cat tree for large cats` 搜索量 7,362，均价 $67.47。
- 40条完整 Cat Tree 样本月销量16,647、月销售额约$870,536，适合用34.88、49.88、69.88等家庭零售价格心智切入。

### Temu：44英寸完整功能款最确定

- 8条经 ProductId 复核的完整 Cat Tree 月销量合计 9,401，月销售额 $246,252.62，加权成交价约 $26.19。
- 两个44英寸完整 Cat Tree 的月销量分别为5,000和2,000，价格约$24，是目前最强的Temu方向证据。
- 59-66英寸款存在成交，但销量明显低于44英寸核心款；首发不应押注75英寸以上超大件。

### AI判断

1. **2026年Q4主战役：**成熟结构的44—54英寸主流款、大猫加强款应在2026年8月上旬至10月上旬分批上架，先承接11月前置峰，再进入12月主峰。主链接不应等到11月才冷启动。
2. **2027年暑期次峰：**净新结构、需要重新开模或重做包装的产品，建议2027年4月8日至5月6日上架，库存最晚2027年5月20日前就绪，承接7月次峰。
3. **关键词差异：**Sorftime关键词季节字段显示 `cat tree tower` 更偏10—12月，`cat tree for large cats` 更偏9月；大猫款应在6—7月提前上架积累评论，普通主流款则重点准备Q4。
4. **常青属性：**`cat tree` 主词被标记为均衡，说明全年都有需求。季节性用于配置新品节奏、库存与广告，而不是在淡季完全停止销售。

### 风险提示与人工复核

- Google Trends数值缺失意味着当前指数主要反映Sorftime的Amazon类目估算，不代表全美所有渠道的真实订单季节性。
- 人工导出Google Trends CSV后，应使用Skill中的 `scripts/analyze_seasonality.py` 重新计算；若Google峰值与Sorftime偏离超过一个月，应下调库存激进度。
- 上架时间是开始积累点击、转化和评论的时间，不是到仓日期；海运、入仓和平台预约需进一步前置。

## 4. 美国客户画像：按购买任务而非虚构人口比例划分

以下画像基于平台商品数据、评论、美国宠物行业背景、公开论坛和社交平台行为交叉推断，不是平台官方用户比例。

### 公寓与租房养猫者

- **购买任务：**在有限空间中提供攀爬、休息和抓挠区域，同时不让客厅显得拥挤。
- **主要阻力：**占地过大、安装复杂、实际尺寸与图片不符、搬家不方便。
- **适配产品：**32-44英寸紧凑款或44-54英寸窄占地款；浅灰、米色、浅木色。
- **平台路径：**Temu / Walmart 更适合入门价位；Amazon 适合强调尺寸透明和稳定性。

### 一至两只成年猫的主流家庭

- **购买任务：**一次购买解决攀爬、躲藏、休息和抓挠，要求价格合理且使用稳定。
- **主要阻力：**晃动、缺件、抓绳脱落、猫窝过小、安装说明不清楚。
- **适配产品：**44-63英寸多层完整 Cat Tree，是项目部最应优先覆盖的主流人群。
- **平台路径：**Amazon / Walmart 双主渠道；Temu 可做简化低价版。

### 大猫与多猫家庭

- **购买任务：**让大体型猫能完整趴卧，多只猫能同时使用，跳跃时结构不晃。
- **主要阻力：**标题写 large cats 但平台、猫窝和底座仍偏小；退货成本高。
- **适配产品：**60-72英寸加强款；宽平台、大入口、粗柱、重底座、防倾倒带。
- **平台路径：**Amazon 最适合用评论与A+建立信任；Walmart适合价值型大猫款。

### 幼猫、老年猫与低跳跃需求

- **购买任务：**用较低台阶、安全坡道和可达平台减少高跳跃压力。
- **主要阻力：**层间距离过大、落脚点错位、平台太滑或太窄。
- **适配产品：**28-36英寸低台阶款，加入坡道、宽顶窝、防滑表面和可拆洗垫。
- **平台路径：**Amazon 适合场景长尾词；Walmart / Temu 适合价格友好版本。

### 家居审美与内容驱动人群

- **购买任务：**猫用品既要能用，也要融入客厅并适合拍照或短视频。
- **主要阻力：**传统灰色绒布款廉价感强；造型款容易牺牲稳定性。
- **适配产品：**复用成熟结构做 pink、flower、cactus、boho、modern wood 换肤。
- **平台路径：**TikTok Shop 用内容验证；Temu 测图测价；Amazon / Walmart 只上验证后的款。

### 核心判断

1. Amazon更适合高证据密度产品：尺寸、稳定、承重、评论和安装内容必须完整。
2. Walmart更适合家庭价值型SKU：价格清楚、交付快、退换方便、客厅场景直观。
3. Temu更适合紧凑和完整功能低价款：先控制外箱、缺件和安装，再谈功能数量。
4. TikTok Shop更适合成熟结构的视觉款，不适合用混合抓板类目数据推导 Cat Tree 市场容量。

### Must-be 必备需求

- 不晃、不倒，提供防倾倒带且安装后有效。
- 平台、猫窝、入口和层间距离与目标猫体型匹配。
- 零件齐全、孔位准确、说明书清晰、附备件。
- 抓绳固定牢固、软包不快速掉毛、板材和边缘安全。
- 包装能够承受大件运输，避免破损和缺件。

### Performance 表现型需求

- 平台越宽、底座越稳、猫窝越适配，转化和满意度越高。
- 安装步骤越少、编号越清楚，退货风险越低。
- 可拆洗垫和可更换抓柱能提高长期价值感。
- 包装体积与产品高度之间的效率直接影响平台适配。

### Attractive 惊喜型需求

- modern wood、boho、pink、flower、cactus等家居化视觉。
- 模块化替换件、备用玩具球、可拆洗软包。
- 真实大猫上树、稳定性演示和安装延时视频。

## 7. 最终产品开发路线

### 1. P1 44-54英寸稳定型主流款（机会分 91）

- **依据：**Amazon清洗样本的核心成交集中在中低价完整猫爬架；实时Top5中54英寸和44英寸款继续强势，Temu 44英寸两款合计月销7,000。
- **结构：**宽底座、1个猫窝、1个吊床、2个主要平台、3-5根抓柱、防倾倒带、零件编号。
- **平台版本：**Amazon $34.99-$49.99；Walmart $34.88-$49.88；Temu 做简化版 $23.99-$29.99。
- **风险：**最容易同质化，必须用稳定、尺寸透明、安装友好和包装完整拉开差异。

### 2. P2 60-72英寸大猫/多猫加强款（机会分 86）

- **依据：**Amazon实时Globlazer 70英寸约$89.99仍有4,648月销量；Walmart large cats关键词均价约$67.47。
- **结构：**平台宽度优先于高度；大猫窝入口、粗柱、重底座、双固定带、15-22lb真实猫实拍。
- **平台版本：**Amazon $79.99-$109.99；Walmart $59.88-$79.88；Temu不作为首批大货。
- **风险：**外箱体积、破损、安装难度和退货成本是立项前硬门槛。

### 3. P3 28-36英寸低台阶细分款（机会分 79）

- **依据：**Temu 32英寸完整款仍有434月销量；Amazon存在小空间、幼猫、老年猫和窗边使用场景。
- **结构：**低层间距、坡道或踏板、宽顶窝、防滑、可拆洗垫、紧凑外箱。
- **平台版本：**Walmart / Temu 做价值款；Amazon用 senior cats、kittens、small spaces 场景词。
- **风险：**容易陷入低价竞争，需要用人群和使用场景而非单纯高度定位。

### 4. P4 成熟结构的审美换肤款（机会分 74）

- **依据：**Amazon实时粉色54英寸款进入头部，说明视觉差异可以与主流结构同时成立；Pinterest/TikTok提供审美灵感但不是销量证据。
- **结构：**只换颜色、软包和局部造型，不牺牲底座、平台和抓柱结构。
- **平台版本：**TikTok Shop / Temu先测内容和点击；验证后进入Amazon / Walmart。
- **风险：**跟卖快、颜色SKU膨胀、实物色差和结构稳定性下降。

## 9. Listing、图片和广告打法

### 共用图片顺序

1. 白底完整产品图，准确展示高度和功能数量。
2. 真实猫体型对比图，尤其是大猫款。
3. 总高、底座、平台、猫窝入口和层间距尺寸图。
4. 稳定结构、防倾倒带、底板和连接点。
5. 抓柱、猫窝、吊床、平台和玩具等功能拆解。
6. 零件编号、安装步骤和安装视频入口。
7. 材料、边缘、抓绳固定和可拆洗部件。
8. 美国公寓客厅或家庭窗边场景。
9. 包装保护、备件包和售后替换流程。

### 平台差异

- **Amazon：**主攻 `cat tree tower`、`cat tree for large cats`、`cat tree for indoor cats` 等明确场景词；主词 `cat tree` 只做受控探索，并持续否定独立抓板、抓柱和玩具噪音。
- **Walmart：**突出价格、WFS/配送、易安装和家庭场景；标题先写高度、适配猫型和核心结构。
- **Temu：**首图强调功能完整和价格感，但必须加入真实猫图与尺寸图，避免图片放大造成预期落差。
- **TikTok Shop：**用猫跳上后不晃、30秒安装、沙发抓挠转移和客厅前后对比做内容；没有纯 Cat Tree 数据前只做小批验证。

## 10. 风险、人工复核与立项门槛

### 风险

- Sorftime销量、销售额和CPC为第三方估算，不等于平台后台订单。
- Amazon实时类目流仍可能混入墙装猫家具，必须保持清洗规则。
- Temu只有8个完整 Cat Tree 复核样本，适合判断方向，不适合推算全站精确市场规模。
- TikTok Shop缺乏纯 Cat Tree 商品级样本，不能输出定量结论。
- 大件物流、破损、缺件和退货可能吞掉表面毛利。

### 样品必须通过

1. 大猫跳跃与多猫同时使用的晃动测试。
2. 防倾倒带与墙体固定有效性测试。
3. 平台、猫窝和入口尺寸实测。
4. 随机抽箱完整安装，验证孔位和缺件率。
5. 抓绳拉脱、软包掉毛、异味和边缘安全测试。
6. 外箱跌落、护角和内部零件固定测试。

## 11. 最终结论

项目部应采用“一套稳定结构平台，按平台拆成三类版本”的开发方式：

1. **Amazon / Walmart主力结构：**44-54英寸稳定主流款。
2. **Amazon利润与品牌结构：**60-72英寸大猫加强款。
3. **Temu验证结构：**32-44英寸紧凑低价款。
4. **TikTok / Temu趋势测试：**在成熟结构上做审美换肤，不另起高风险复杂结构。

不建议首批开发独立抓板、墙面踏板或75英寸以上超高塔来替代本次 Cat Tree 主线。它们属于相邻品线或高风险扩展，不计入本报告的核心选品结论。
